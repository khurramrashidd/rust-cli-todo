# 📋 Rust CLI To-Do List

A simple **command-line To-Do List application** written in Rust.
You can add, view, and delete tasks directly from the terminal.

---

## 🚀 Features

* Add new tasks
* View all tasks
* Delete tasks
* Save tasks to file
* Persistent storage using `tasks.txt`

---

## 🦀 Built With

* Rust
* Standard library (no external crates)

---

## 📂 Project Structure

```
todo_app
│
├── Cargo.toml
├── tasks.txt
└── src
    └── main.rs
```

---

## ⚙️ Installation & Run

### 1️⃣ Clone repository

```
git clone https://github.com/YOUR_USERNAME/rust-cli-todo.git
```

### 2️⃣ Navigate to project

```
cd rust-cli-todo
```

### 3️⃣ Run the app

```
cargo run
```

---

## 🖥 Example Usage

```
===== TODO APP =====
1. Show Tasks
2. Add Task
3. Delete Task
4. Exit
```

Example:

```
Enter new task:

```

Output:

```
Your Tasks:
```
